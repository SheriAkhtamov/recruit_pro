# Вклад в RecruitmentTracker

Спасибо за интерес к проекту! Этот документ поможет вам начать внесение изменений.

## 🚀 Как начать

1. **Fork репозитория**
   - Нажмите "Fork" на странице GitHub
   - Клонируйте ваш fork: `git clone <your-fork-url>`

2. **Создайте ветку**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Установите зависимости**
   ```bash
   npm install
   ```

4. **Настройте окружение**
   ```bash
   cp .env.example .env
   # Настройте .env с вашими параметрами
   ```

## 📝 Правила разработки

### Код стиль
- Используйте TypeScript для всех новых файлов
- Следуйте существующим соглашениям об именовании
- Добавляйте комментарии для сложной логики

### Коммиты
- Используйте осмысленные сообщения коммитов
- Формат: `type(scope): description`
  - `feat`: новая функциональность
  - `fix`: исправление багов
  - `docs`: документация
  - `style`: форматирование кода
  - `refactor`: рефакторинг
  - `test`: тесты

### Pull Request
- Создайте PR из вашей ветки в `main`
- Добавьте описание изменений
- Убедитесь что все тесты проходят
- Дождитесь ревью

## 🧪 Тестирование

```bash
# Проверка типов
npm run check

# Сборка проекта
npm run build

# Запуск в режиме разработки
npm start
```
